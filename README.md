# BE_Item_Server

## 🌐 프로젝트 개요

해당 프로젝트는 DPANG 서비스의 상품 서버로서, 사용자의 상품, 상품 문의, 인기 상품과 관련한 서비스를 담당하고 있습니다.

## 🛠️ 프로젝트 개발 환경

프로젝트는 아래 환경에서 개발되었습니다.

> OS: MacOS Sonoma 14.2.1
> IDE: Intellij IDEA
> Java 17
